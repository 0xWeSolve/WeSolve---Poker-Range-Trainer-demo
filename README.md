# WeSolve — Preflop Range Trainer

Train your preflop ranges offline, no account, no subscription. A single HTML file — open it in your browser and start.

## Features

- Build and save opening & defense ranges
- Smart Quiz that weights hands you hesitate on
- Ghost overlay to compare your range vs GTO reference
- Import/Export with frequency support
- 3 skins, 5 color themes, 4 UI styles
- Built-in NL25 ranges to get started
- Radio 🎵

## How to use

1. Download `WeSolve.html`
2. Open it in any browser
3. That's it — no install, no account, works offline
